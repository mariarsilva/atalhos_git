Atalhos importantes no Git:

cd = acessar pasta cd .. = voltar um diretório git init = iniciar diretório onde deseja salvar seu projeto mkdir = cria diretório na pasta desejada git congif --list = mostra as configurações salvas no momento branch = versões do sistema. A versão principal se chama master git status = vendo o status da pasta git add = adicionar os arquivos git add -A = adiciona todos os arquivos ainda não adicionados git commit -m "frase" = faz commit e insere o texto do commit git log = ve o log de alterações
